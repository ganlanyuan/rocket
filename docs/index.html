Rocket is a powerful Sass framework with many common used components and utility functions, to help you build websites faster and easier.

  • Responsive

    Rocket is designed to working with different medias and devices. With Rocket, it's easy to make your website responsive.

  • Customizable

    Rocket is fully customizable. No required class or id names. @include the components with your customized options.

  • light weight

    Unlike other frameworks which give you a lot of code that you will never use, Rocket gives you only the part you request. No request, no code!

  • Accessible

    Rocket components come with ARIA attributes and JavaScript utilities to make it fully accessible for everyone.

Get started

  1. Download Rocket

    Download Rocket via npm, bower or download the source files directly from the github page.

  2. Import Rocket and Start using Rocket Mixins and Functions

    Import Rocket into your sass file. (More details about Sass import, visit http://sass-lang.com/guide#topic-5)

    You can now use all the mixins and functions.

    scss 
    @import "path/to/rocket.scss";

.wrapper {
  @include grid(3 7);
}

  3. Compile sass to css

    Use one of these tools or a Task runner to compile your sass files to css files.
