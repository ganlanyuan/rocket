  • Layout setting

    Default container and gutter for container, grid, gallery, liquid-2, liquid-3, metro, masonry.

    • scss 
      $ro-layout: (
  container: 1024px,
  gutter: 20px,
);

  • Container

    Create a container to constrain your content. It can be pushed to the left, right edge or center aligned.

    IE 7+, Firefox 4+, Chrome 19+, Safari 6+, Opera 15+

    scss: syntax 
    @mixin container($container ('gutter' $gutter) $alignment)
// $container: px | em | %
// $gutter: px | em | %
// $alignment: left | right | center
    • markup 
      <div class="container1 " data-contentbox>
  <!-- contents -->
</div>
    • scss 
      .container1 { @include container(600px); }
    • markup 
      <div class="container2 " data-contentbox>
  <!-- contents -->
</div>
    • scss 
      .container2 {
  @include container(800px gutter 2% left);
}

  • Grid

    Create a responsive multi columns (and even multi rows) layout using grid.

    IE 8+, Firefox 28+, Chrome 21+, Safari 6.2+, Opera 15+

    scss: syntax 
    @mixin grid($grid (gutter $gutter) (child $child) $breakpoints $condition $media-type $direction keep)
// $grid: list | map
// $condition: 'min' | 'max'
// $media-type: media types
// $breakpoints: map, e.g. (small: 640px, medium: 768px, large: 1024px)
// $direction: LTR | RTL
    • markup 
      <div class="grid1">
  <div data-contentbox></div>
  <div data-contentbox></div>
  <div data-contentbox></div>
</div>
    • scss 
      .grid1 { @include grid( (3 7 4) ); }
// 1st div child: 3 columns;
// 2nd div child: 7 columns;
// 3rd div child: 4 columns;

    You can change the column orders using (cols : order) format. Here the order works the same as flexbox-order works: https://drafts.csswg.org/css-flexbox-1/#order-property

    • markup 
      <div class="grid2">
  <div data-contentbox>1</div>
  <div data-contentbox>2</div>
  <div data-contentbox>3</div>
</div>
    • scss 
      .grid2 { @include grid( (3 7 4 : 1 0 0) ); }
// or (3:1, 7:0, 4:0)
// orders: 1 0 0. The first div child will go last.

    3

    The child elements can be separated to multi rows using a nested list format: ( (row1 row1 row1) (row2 row2) (row3 row3 row3)).

    • markup 
      <div class="grid3">
  <div data-contentbox>1</div>
  <div data-contentbox>2</div>
  <div data-contentbox>3</div>
  <div data-contentbox>4</div>
  <div data-contentbox>5</div>
</div>
    • scss 
      $main: ( (3 7 4) (2 5) );
.grid3 { @include grid($main); }
// 1st row: (3 7 4)
// 2nd row: (2 5)

    • markup 
      <div class="grid4">
  <section data-contentbox>1</section>
  <section data-contentbox>2</section>
</div>
    • scss 
      .grid4 { @include grid( (2 5) child section); }

    Grid works with breakpoints to make multi layouts for different window size.

    • markup 
      <div class="grid5">
  <div data-contentbox>1</div>
  <div data-contentbox>2</div>
  <div data-contentbox>3</div>
  <div data-contentbox>4</div>
  <div data-contentbox>5</div>
</div>
    • scss 
      $breakpoints: (
  medium: 800px,
  large: 1200px,
);
$main: (
  'default': (3 4) ( 2 5) 1,
  medium: (2 7 3: 1 0 1) (1 1: 1 0),
  1000px: (2 7 3 4 4),
);
.grid5 { @include grid($main $breakpoints); }

  • Liquid-2

    Make a fluid 2-columns blog layout, with a fixed sidebar and fluid main column.

    IE 8+, Firefox 4+, Chrome 19+, Safari 6+, Opera 15+

    scss: syntax 
    @mixin liquid-2($layout (gutter $gutter) (child $child) $breakpoints $condition $media-type $direction)
// $layout: list | map
// $condition: 'min' | 'max'
// $media-type: media types
// $breakpoints: map, e.g. (small: 640px, medium: 768px, large: 1024px)
// $direction: LTR | RTL
    • markup 
      <div class="liquid-2-1">
  <div data-contentbox>1</div>
  <div data-contentbox>2</div>
</div>
    • scss 
      .liquid-2-1 { @include liquid-2( ('auto' 200px) ); }
// 1st column: fluid;
// 2nd column: 200px;
    1
    2

    The order and gutter can be redefined as well.

    • markup 
      <div class="liquid-2-2">
  <div data-contentbox>1</div>
  <div data-contentbox>2</div>
</div>
    • scss 
      .liquid-2-2 {
  @include liquid-2( ('auto':2, 200px:1) gutter 10px);
}
// order: 2 1;

    • scss

@include bp('min' 'height' 640) {};
